## Setup your environment

Create a `.env` file and add the following environment variables

```plain
REACT_APP_GOOGLE_SHEET_URL=https://docs.google.com/spreadsheets/d/<docid>/gviz/tq?tqx=out:csv
MONGODB_URI=
BACKEND_PORT=
```
